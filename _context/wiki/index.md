## Quick summary

This is the **HashiCorp Terraform CLI** — the source-available infrastructure-provisioning tool. Work spans bug fixes, new features, performance, and refactoring, with the heaviest focus on `internal/terraform` and closely related packages (`addrs`, `configs`, provider protocol, CLI commands).

See [project.md](project.md) for architecture detail and [preferences.md](preferences.md) for how to work effectively on this codebase.
