# `floor` Function

`floor` returns the closest whole number that is less than or equal to the
given value, which may be a fraction.

## Examples

```
> floor(5)
5
> floor(4.9)
4
```

## Related Functions

* [`ceil`](./ceil.html), which rounds to the nearest whole number _greater than_
  or equal.
