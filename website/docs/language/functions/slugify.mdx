# `slugify` Function

`slugify` takes two values: a separator and a string. It returns a slug of the original string separated by the specified separator.

```hcl
slugify(separator, string)
```

## Examples

```
> slugify("-", ""HellO#$?World!")
hello-world

> slugify("_","abc 123 abc 456 _")
abc_123_abc_456
```
